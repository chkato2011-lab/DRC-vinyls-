⚠ Sin acceso a Discogs API
Esta app no puede conectarse a Discogs desde dentro de claude.ai.
Solución: Descargá el archivo HTML → abrilo directo en Chrome/Firefox como archivo local (file://) → ahí todo funciona.
Paso 1 — Obtener token de Discogs
1. Entrá a discogs.com → Configuración → Desarrolladores
2. Hacé clic en "Generate new token"
3. Copiá el token generado
4. Pegalo arriba en ▶ Token Discogs y guardá

Paso 2 — Exportar tu colección
1. Entrá a discogs.com → Mi colección
2. Clic en "Exportar" (esquina superior derecha)
3. Descargá el archivo CSV cuando esté listo

Paso 3 — Importar
Subí el CSV acá abajo. Busca por Catalog# primero, si falla busca por Title. Si hay múltiples versiones te pregunta cuál elegir.
Los fallidos quedan guardados en "Discos no encontrados" en la pestaña "Por código".
⏱ Se procesa de a uno para respetar el límite de la API (~3-4 seg por disco).
