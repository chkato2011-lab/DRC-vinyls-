// desde discogsIMPORTAR
Necesitás un Personal Access Token de Discogs para importar.
Cómo obtenerlo: Entrá a discogs.com → Configuración → Desarrolladores → Generate new token → copialo y pegalo acá.
Paso 1 — Obtener token de Discogs
1. Entrá a discogs.com → Configuración → Desarrolladores
2. Hacé clic en "Generate new token"
3. Copiá el token generado
4. Pegalo arriba en Token Discogs y guardá

Paso 2 — Exportar tu colección de Discogs
1. Entrá a discogs.com → Mi colección
2. Clic en "Export" (esquina superior derecha)
3. Descargá el archivo CSV cuando esté listo

Paso 3 — Importar
Buscá por código de catálogo. Si hay múltiples versiones te pregunta cuál elegir. Los fallidos quedan guardados en "Discos no encontrados".
⏱ Se procesa de a uno para respetar el límite de la API (~3-4 seg por disco).
